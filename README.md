---

## 🎯 Objetivo
Desenvolver, em equipe, um aplicativo mobile de agendamentos online utilizando React Native, com foco na construção das interfaces (UI) e navegação entre telas, utilizando dados mockados (fictícios).

---

## 📱 Descrição do Projeto
O **Agendei** é um aplicativo que simula um sistema de agendamento de serviços, como consultas médicas, estética e outros.

Nesta versão, o aplicativo não possui integração com backend, sendo totalmente baseado em dados simulados.

---

## 🖼️ Telas do Aplicativo

O app contém as seguintes telas:

- Splash (Tela Inicial)
- Login
- Home (Categorias de serviços)
- Explorar (Lista de estabelecimentos/profissionais)
- Reservas (Agendamentos do usuário)
- Perfil
- Estabelecimento (Detalhes e serviços)
- Agendamento (Seleção de data e horário)

---

## 🛠️ Tecnologias Utilizadas

- React Native
